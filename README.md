# Hi, I'm Joan Caceres

I build Python tools and research code around quantum circuits, superconducting devices, simulation workflows, and scientific computing.

## What I Work On

- Superconducting and hybrid quantum circuit simulation
- Quantum hardware design workflows
- Scientific Python tools for analysis, modeling, and automation
- Layout and CAD utilities for research workflows

## Featured Projects

- [cqedraw](https://github.com/joanjcaceres/cqedraw) - Browser and desktop GUI for drawing superconducting circuit graphs and generating matrix snippets for Black Box Quantization workflows.
- [HybridSuperQubits](https://github.com/joanjcaceres/HybridSuperQubits) - Python framework for simulating hybrid semiconductor-superconductor quantum circuits.
- [klayout-duplication-cleaner](https://github.com/joanjcaceres/klayout-duplication-cleaner) - Scripts for detecting and removing duplicate shapes in KLayout layouts.
- [three-body-problem](https://github.com/joanjcaceres/three-body-problem) - A compact Python simulation of the three-body problem.
